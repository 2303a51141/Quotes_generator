# 💬 Quotes Generator

A simple and responsive web application that displays inspiring random quotes with a clean and modern user interface. Users can generate new quotes instantly with a single click, making it perfect for daily motivation and inspiration.

## 🌐 Live Demo

🔗 https://2303a51141.github.io/Quotes_generator/

---

## ✨ Features

- 🎲 Generate random inspirational quotes
- ⚡ Instant quote generation
- 📱 Responsive design for all devices
- 🎨 Clean and modern UI
- 🚀 Fast and lightweight
- 💻 Easy to use

## 🚀 Getting Started

### Clone the repository

```bash

```

### Navigate to the project folder

```bash
cd Quotes_generator
```

### Run the project

Simply open **index.html** in your browser.
